# Feature Proposals

This directory contains proposals for new Forge features and enhancements.

## Creating a Proposal

1. Copy `TEMPLATE.md` to a new file: `feature-name.md` (e.g., `parallel-task-execution.md`)
2. Fill in all sections
3. Submit a PR for review

## Proposal Lifecycle

| Status | Description |
|--------|-------------|
| **Draft** | Initial idea, not yet ready for review |
| **Under Review** | Open for feedback and discussion |
| **Accepted** | Approved for implementation |
| **Rejected** | Not moving forward (with rationale) |
| **Implemented** | Feature has been built and merged |
